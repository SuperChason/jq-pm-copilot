# 页面模式

## 标准框架

- 侧栏固定宽 200px，从顶部到底部。
- 顶栏高 60px，位于侧栏右侧。
- 主内容区背景 #F5F5F7，内边距 12px。
- 页面卡片白底、4px 圆角、16px 内边距和轻阴影。

## 列表管理页

页面标题与主操作位于顶部。筛选区采用常用条件直出、低频条件折叠；工具栏承载批量操作；表格负责数据浏览；分页紧随表格。保留无数据、无搜索结果、加载失败和无权限状态。

## 详情与单据页

顶部展示状态、编号和主操作。主体按业务主题分组，关键信息优先；操作历史、审批轨迹和附件放在稳定位置。编辑模式与查看模式明确区分。

## 表单页

短表单单列，复杂表单按业务主题分组。必填标记、帮助、校验和错误信息靠近字段。离开未保存页面前给出提示。

## 仪表盘

先呈现关键指标和异常，再给趋势、结构和明细入口。图表必须有标题、时间范围、单位、口径和数据更新时间，颜色含义保持一致。

## 待办页

左侧可放任务分类，右侧为查询、列表和处理入口。突出任务状态、紧急程度、截止时间和发起人。处理完成后明确反馈下一条任务或返回路径。

## 权限与空状态

无权限说明原因和申请路径。首次使用空状态解释价值和下一步；搜索空状态保留筛选条件并提供清除入口；系统错误提供重试和问题反馈方式。
